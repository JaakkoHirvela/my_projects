pankkivapaa selvittää käyttäjälle onko annettu päivämäärä pankkivapaapäivä. Jos on, antaa käyttäjälle seuraavan pankkipäivän.
Käännä ja aja QtCreatorilla. Yksinkertainen pdf-demo myös saatavilla.

pankkivapaa checks if given date is a bank holiday. If it is, then the app prints out the next free bank day.
Build and run with QtCreator. Simple pdf-demo also available.

Alusta/platform: Windows 10 64-bit.
tekijä/author: Jaakko Hirvelä
